# Maintainer Notes

## What this project is trying to become
A trustworthy, regularly updated map of the AI agent ecosystem.

## What would make it worse
- bloating the list with every barely-different tool
- copying vendor marketing language
- mixing products, frameworks, and infrastructure without distinction
- rewarding noise over accuracy

## Maintenance priorities
1. keep the repo useful
2. keep the repo accurate enough to trust
3. keep the repo legible for newcomers
4. keep the standards high enough that contributors improve it instead of diluting it

## Merge philosophy
Prefer fewer, better additions over rapid list inflation.

## Reviewer shortcut
A good addition should answer:
- what is it?
- who is it for?
- why is it notable?
- where is the official source?
