# Support

## What this repo is
AI Agent Landscape is a curated content project.

It does not provide a runtime, hosted API, package manager, or software support SLA.

## Best ways to contribute or ask for changes
- use the issue templates for additions and corrections
- open a pull request if you already know the exact fix
- include official sources when possible

## What maintainers can help with
- adding missing tools
- correcting categories
- fixing stale descriptions, pricing, or licensing notes
- improving clarity and structure

## What maintainers cannot do
- provide support for third-party tools listed in the repo
- debug vendor products
- validate every marketing claim without sources

## Quality bar
High-signal issues and PRs get attention faster than broad or vague requests.
