# Contributing to AI Agent Landscape

Thanks for helping improve the map.

## What to contribute
- new agent entries
- corrections to pricing, licensing, or platform support
- better categorization
- broken links
- clearer descriptions
- comparison criteria improvements

## Contribution rules
- be specific
- link to official sources when possible
- avoid pure marketing language
- if a claim is subjective, label it as opinion
- if a tool spans multiple categories, explain why

## Entry format
When suggesting a new tool, include:
- name
- homepage / repo
- category
- open source or closed
- best-for summary
- pricing model
- local / cloud / hybrid
- key capabilities
- brief note on strengths and limits

## Style
Write like a helpful operator, not a press release.

Good:
- "Strong for browser automation, weaker for local desktop tasks."

Bad:
- "Revolutionary next-generation AI solution for all your needs."

## Pull requests
Small, clean PRs are preferred.
If you are adding many entries, split them by category if possible.
