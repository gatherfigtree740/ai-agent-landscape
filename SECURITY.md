# Security Policy

## Scope
This repository is primarily a curated content and data project.

It currently contains:
- Markdown documentation
- SVG visual assets
- CSV dataset entries
- GitHub issue templates

It does **not** ship runtime code, install scripts, package managers, CI workflows, browser extensions, or background services.

## Safety note
That means the repo is intentionally low-risk to browse and clone.

There are currently:
- no GitHub Actions workflows
- no executable setup scripts
- no package postinstall hooks
- no environment secrets committed to the repo

## Reporting a concern
If you spot something risky, misleading, or suspicious:
- open a private security report if the platform supports it
- or open an issue describing the concern without posting secrets

## Please do not submit
- API keys
- access tokens
- passwords
- private credentials
- proprietary data you do not have permission to share

## Maintainer policy
Security and trust beat speed.
Questionable contributions, surprise automation, or unsafe third-party additions should not be merged just to increase activity.
